To activate ASCII Saver:

  1. Open System Settings → Screen Saver
  2. Select ASCII Saver from the list
  3. Launch ASCII Saver Camera Agent from Applications
  4. Grant camera permission when prompted

The camera agent will appear as a small icon in your menu bar. It only captures when the screensaver is active.

Tip: Add the camera agent to System Settings → General → Login Items so it starts automatically.
